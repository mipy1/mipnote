### Usage ğŸ¼:

To configure mipnote, edit the `mipnote.conf` file

To create a new note, or edit an existing one:
```bash
mipnote
```

To initialise a notes repository:
```bash
mipnote -i # OR --init
```

To get a preview of a note:
```bash
mipnote -p # OR --preview
```

To get help:
```bash
mipnote -h # OR --help
```
